# BookMarks

## サイト概要
ブックマークの管理を行い、利便性を高める。
ブックマークは他人と共有できる。

### サイトテーマ
ブックマークしたサイトを見やすく整理できる。

### テーマを選んだ理由
便利にしたかったため。
ブックマークしたサイトが何のサイトであったか忘れがちであったため、
登録したサイトをひと目見ただけでわざわざ開かなくても確認できるようにしたかったため。


###AI機能􏰁概要　専門技術講座AI教養コース
自分がブックマークしているサイトから判断して、新たにおすすめのサイトを出す。


### ターゲットユーザ
自分。ブックマークを頻繁に行う方。

### 主な利用シーン
ブックマークの管理を行い、利便性を高める。
